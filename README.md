# goit-rdb-hw-02

Проєкт: Normalization homework (GOIT)
Мета: Нормалізувати початкову таблицю до 1НФ/2НФ/3НФ, створити ER-діаграму та таблиці у MySQL.
Коротко: Вихідна таблиця містила повторювані та неправильно організовані дані (список товарів в одному полі). Було створено таблиці customers, orders, products, order_items відповідно до 3НФ. SQL-скрипт додається.
